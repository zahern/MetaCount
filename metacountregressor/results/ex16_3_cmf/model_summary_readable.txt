  Model: log(crashes) = Intercept + b_AADT·log(AADT)
       + sum [Upper terms: direct effect on crash rate]
       + sum [Lower terms: modify AADT elasticity]
       + log(segment length)  [exposure offset]

  CORE PARAMETERS
  --------------------------------------------------------------------
  Intercept                                        -6.7188
  log(AADT) elasticity                             +0.8517

  Lane Width (ft)                                  -0.2784      -0.0172 (per SD)  CMF= 0.9829

  LOWER-LEVEL TERMS  (modify AADT elasticity — x × log(AADT) interaction)

  These do NOT affect the baseline crash rate directly.  They shift
  how steeply crash risk scales with AADT:  a negative coefficient means
  segments with high values of that variable are LESS sensitive to AADT.

  Variable                                            Coef  Interpretation
  --------------------------------------------------------------------
  Average Annual Precipitation (in/yr)             +0.0102  => higher AADT sensitivity
  Maximum Median Shoulder Width (ft)               +0.0111  => higher AADT sensitivity
  Average Longitudinal Slope (%)                   +0.1247  => higher AADT sensitivity

  Base AADT elasticity (log_aadt coeff): +0.8517
  Local elasticity_i = beta_AADT + sum(gamma_k * x_k_std_i)
  'Monotonic AADT' constraint: elasticity > 0 for EVERY segment in dataset.
  Meaning: more traffic always -> more crashes (no segment ever reverses).
