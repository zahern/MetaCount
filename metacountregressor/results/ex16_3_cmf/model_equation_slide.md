### What each coefficient means for crash prediction

> **How to read this table:** The model predicts $\log(\text{crashes}) = $ sum of all terms below. Each row shows the exact contribution for one variable.

| Term                                           | Coefficient   | Variable / Formula                                                                              | Interpretation                                                                                                                                                                             |
| ---------------------------------------------- | ------------- | ----------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Intercept** $\alpha$                         | -6.7188       | Baseline (no traffic, no geometry adjustments)                                                  | $e^{-6.719} = 0.00$ crashes per unit exposure at reference conditions                                                                                                                      |
| **Base AADT elasticity** $\beta_{\text{AADT}}$ | +0.8517       | $\beta \times \log(\text{AADT})$                                                                | Each 1% more traffic -> crashes change by 0.85% (elasticity = 0.852)                                                                                                                       |
| ---                                            | ---           | **UPPER TERMS — direct effect, same CMF at any AADT**                                           |                                                                                                                                                                                            |
| $\beta_{CURVES}$                               | -0.0766       | $\beta \times \text{Number of Horizontal Curves (count)}$ (per 1 SD change)                     | CMF = $e^{-0.0766} = 0.9262$ (safer, 7.4% reduction)                                                                                                                                       |
| $\beta_{GBRPM}$                                | +0.0446       | $\beta \times \text{Grade-Break Rate per Mile}$ (per 1 SD change)                               | CMF = $e^{0.0446} = 1.0456$ (higher risk, 4.6% increase)                                                                                                                                   |
| $\beta_{MEDWIDTH}$                             | +0.0315       | $\beta \times \text{Median Width (ft)}$ (per 1 SD change)                                       | CMF = $e^{0.0315} = 1.0320$ (higher risk, 3.2% increase)                                                                                                                                   |
| $\beta_{MIGRADE}$                              | -0.0714       | $\beta \times \text{Minimum Vertical Grade (%)}$ (per 1 SD change)                              | CMF = $e^{-0.0714} = 0.9310$ (safer, 6.9% reduction)                                                                                                                                       |
| $\beta_{SLOPE}$                                | -1.2186       | $\beta \times \text{Average Longitudinal Slope (%)}$ (per 1 SD change)                          | CMF = $e^{-1.2186} = 0.2956$ (safer, 70.4% reduction)                                                                                                                                      |
| $\beta_{SPEED}$                                | -0.0851       | $\beta \times \text{Posted Speed Limit (mph)}$ (per 1 SD change)                                | CMF = $e^{-0.0851} = 0.9184$ (safer, 8.2% reduction)                                                                                                                                       |
| $\beta_{WIDTH}$                                | -0.2784       | $\beta \times \text{Lane Width (ft)}$ (per 1 SD change)                                         | CMF = $e^{-0.2784} = 0.7570$ (safer, 24.3% reduction)                                                                                                                                      |
| ---                                            | ---           | **LOWER TERMS — modify AADT elasticity (traffic interaction)**                                  |                                                                                                                                                                                            |
| $\gamma_{AVEPRE}$                              | +0.0102       | $\gamma \times \text{Average Annual Precipitation (in/yr)} \times \log(\text{AADT})$ (per 1 SD) | Amplifies AADT elasticity by 0.0102 (per 1 SD). Higher values make crash rate grow faster with AADT.                                                                                       |
| $\gamma_{MXMEDSH}$                             | +0.0111       | $\gamma \times \text{Maximum Median Shoulder Width (ft)} \times \log(\text{AADT})$ (per 1 SD)   | Amplifies AADT elasticity by 0.0111 (per 1 SD). Higher values make crash rate grow faster with AADT.                                                                                       |
| $\gamma_{SLOPE}$                               | +0.1247       | $\gamma \times \text{Average Longitudinal Slope (%)} \times \log(\text{AADT})$ (per 1 SD)       | Amplifies AADT elasticity by 0.1247 (per 1 SD). Higher values make crash rate grow faster with AADT.                                                                                       |
| ---                                            | ---           | **OFFSET — exposure correction**                                                                |                                                                                                                                                                                            |
| **Offset**                                     | 1.000 (fixed) | $+ \log(\text{Segment Length, mi})$                                                             | Longer segments have proportionally more crashes. Doubling length doubles expected crashes (holding rate constant). This makes coefficients comparable across segments of different sizes. |

> **Full equation:** $\log(\hat{\mu}_i) = \alpha + \beta_{\text{AADT}} \log A_i + \sum_j \beta_j x_{ij} + \sum_k \gamma_k x_{ik} \log A_i + \log L_i$
