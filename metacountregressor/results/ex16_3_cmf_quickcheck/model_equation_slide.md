### What each coefficient means for crash prediction

> **How to read this table:** The model predicts $\log(\text{crashes}) = $ sum of all terms below. Each row shows the exact contribution for one variable.

| Term                                           | Coefficient   | Variable / Formula                                                                                             | Interpretation                                                                                                                                                                             |
| ---------------------------------------------- | ------------- | -------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Intercept** $\alpha$                         | -8.0711       | Baseline (no traffic, no geometry adjustments)                                                                 | $e^{-8.071} = 0.00$ crashes per unit exposure at reference conditions                                                                                                                      |
| **Base AADT elasticity** $\beta_{\text{AADT}}$ | +0.9856       | $\beta \times \log(\text{AADT})$                                                                               | Each 1% more traffic -> crashes change by 0.99% (elasticity = 0.986)                                                                                                                       |
| ---                                            | ---           | **UPPER TERMS — direct effect, same CMF at any AADT**                                                          |                                                                                                                                                                                            |
| $\beta_{MIMEDSH}$                              | -0.1778       | $\beta \times \text{Minimum Median Shoulder Width (ft)}$ (per 1 SD change)                                     | CMF = $e^{-0.1778} = 0.8371$ (safer, 16.3% reduction)                                                                                                                                      |
| $\beta_{MXMEDSH}$                              | +0.2224       | $\beta \times \text{Maximum Median Shoulder Width (ft)}$ (per 1 SD change)                                     | CMF = $e^{0.2224} = 1.2491$ (higher risk, 24.9% increase)                                                                                                                                  |
| $\beta_{WIDTH}$                                | -0.2009       | $\beta \times \text{Lane Width (ft)}$ (per 1 SD change)                                                        | CMF = $e^{-0.2009} = 0.8180$ (safer, 18.2% reduction)                                                                                                                                      |
| ---                                            | ---           | **LOWER TERMS — modify AADT elasticity (traffic interaction)**                                                 |                                                                                                                                                                                            |
| $\gamma_{FRICTION}$                            | -0.0055       | $\gamma \times \text{Pavement Friction, Skid Number (higher = more grip)} \times \log(\text{AADT})$ (per 1 SD) | Dampens AADT elasticity by 0.0055 (per 1 SD). Higher values reduce how fast crash rate grows with AADT.                                                                                    |
| ---                                            | ---           | **OFFSET — exposure correction**                                                                               |                                                                                                                                                                                            |
| **Offset**                                     | 1.000 (fixed) | $+ \log(\text{Segment Length, mi})$                                                                            | Longer segments have proportionally more crashes. Doubling length doubles expected crashes (holding rate constant). This makes coefficients comparable across segments of different sizes. |

> **Full equation:** $\log(\hat{\mu}_i) = \alpha + \beta_{\text{AADT}} \log A_i + \sum_j \beta_j x_{ij} + \sum_k \gamma_k x_{ik} \log A_i + \log L_i$
